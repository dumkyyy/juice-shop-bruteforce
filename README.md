# Juice Shop BruteForce


Инструмент для автоматического перебора паролей на OWASP Juice Shop.


## Возможности

- Одиночный режим - перебор паролей для одного email (-s) 
- Массовый режим - перебор для списка email из файла (-m)


## Установка

1. Убедитесь, что установлен Python 3.6+
2. Установите необходимые зависимости(requirements.txt):
    ```bash
    pip install -r requirements.txt



## Использование

python3 juice_brute.py -u [URL цели] -p [PORT цели] -s/-m [Почту/файл с почтами] [Файл с паролями]
